Introdução ao Trampolim

O que é o Trampolim?

O Trampolim é uma plataforma de empregabilidade inclusiva desenvolvida para conectar profissionais com deficiência ou neurodivergências a empresas preparadas. Nosso foco é eliminar barreiras na contratação.

O que o sistema faz?

Em vez de focar apenas em competências técnicas, nosso sistema cruza os seus requisitos assistivos (como home office, salas silenciosas ou rampas) com a real infraestrutura das empresas parceiras.

Acessibilidade em Primeiro Lugar

Toda a nossa interface foi desenhada para ser simples. Oferecemos ferramentas integradas para ajustar o tamanho da letra e fontes adaptadas para dislexia, garantindo uma navegação confortável para todos.
